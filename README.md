# Lua-bin

`lua-bin` is a repository for building Lua binaries on Windows.

## Lua Version

`lua-5.4.8`

## Build Instructions

1. Double-click `build.cmd` to automatically generate and build the project.  
2. After building, the generated executable files will be available in the project output directory.

## Dependencies

> Lua - [www.lua.org](https://www.lua.org)  
> Premake - [premake.github.io](https://premake.github.io)

## Notes

- This repository is intended solely for generating Lua binaries on Windows.  
- Pre-built releases are not yet available and will be provided once I have confirmed the project structure is correct.
