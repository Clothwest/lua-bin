# Lua-bin

`lua-bin` is a repository for building Lua binaries on Windows.

## Bin

`bin` folder contains files for version [5.4.8][5.4.8].

## Supported Lua Version

[lua-5.4.8][5.4.8]

## Dependencies

> Lua - [www.lua.org](https://www.lua.org)  
> Premake - [premake.github.io](https://premake.github.io)

[5.4.8]: https://github.com/Clothwest/lua-bin/tree/lua-5.4.8
